Démo statique ELIPS

Contenu du dossier :
- index.html
- styles.css
- app.js

Usage local :
- Ouvrir index.html dans un navigateur
ou
- Servir le dossier avec un petit serveur statique

Publication GitHub :
- pousser le contenu de ce dossier dans un repo
- possible sur GitHub Pages

Note :
- la navigation entre pages est simulée côté JS
- pour GitHub Pages, l'approche la plus simple est d'utiliser la racine avec index.html
